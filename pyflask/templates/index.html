The data used in this website is extracted from Facebook using its Marketing API. Only the monthly active users on facebook are considered for the population count. The data is extracted for various target groups based on gender, age group, scholarities and the operating system they use.


Based on existing research by scientists, the type of operating system used by the user can be correlated to his/her income. Users with greater income tend to have iOS device compared to those with a lower income. So we associate the OS type iOS with a greater income and android with a lower income.
