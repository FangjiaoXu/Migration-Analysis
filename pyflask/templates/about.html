About this website


This website was created with the aim to visualize and explore data related to migration collected from the Facebook Marketing API. This website does not offer official statistics on international migration.

Please check out the Project Details below for more information.

This website aims to visualize data related to migration collected from the Facebook Marketing API. This website does not offer official statistics on international migration. The data used for the project was collected through the Facebook Marketing API. More specifically, data for the estimated number of monthly active users (MAUs) matching certain targeting criteria was collected. PySocialWatcher, a package for social data collection built on Python, was used for collecting the data. The data was collected for 231 hosting countries and territories and for 89 countries and territories of origin. See the full list in the FAQ. Using the collected data, we created visualisations using Python libraries such as matplotlib and folium.
