About the website


This website was created with an aim to visualise the migration across various countries based on data collected from the Facebook Marketing API.
Check out the Project Details for more information.

This website aims to visualise the flow of migration around the world. The data used for the project was acquired from the Facebook Marketing API. Pysocialwatcher, a package for social data collection built on python, was used for collecting the data. The data was collected separately for 231 countries with information regarding expats from 89 different places. Using the same data, we created visualisations using some famous python libraries like matplotlib and folium.

Frequently Asked Questions




The data used in this website is extracted from Facebook using its Marketing API. Only the monthly active users on facebook are considered for the population count. The data is extracted for various target groups based on gender, age group, scholarities and the operating system they use.


Based on existing research by scientists, the type of operating system used by the user can be correlated to his/her income. Users with greater income tend to have iOS device compared to those with a lower income. So we associate the OS type iOS with a greater income and android with a lower income.
