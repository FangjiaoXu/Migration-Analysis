Concerning current places of residence, we have collected data for 231 hosting countries and territories with the full list available at here. These are virtually all internationally recognized countries and territories with the main exceptions being Cuba, Syria, North Korea, Sudan, and Northern Cyprus. We collected data for all countries and territories supported by Facebook’s Marketing API at the time of collection (June 2019).

Concerning countries of origin, we have collected data for 89 countries and territories of origin supported by Facebook. See here for the full list. Note that certain sending countries with a large number of migrants living abroad, such as China, Egypt, Syria or Turkey, are not supported by Facebook’s Marketing API.


Facebook provides advertisers with the possibility to selectively show advertisement to Facebook users matching certain targeting criteria. The criteria supported by Facebook’s platform include information on countries a Facebook user has lived in. So an advertiser could, for example, choose to show an advertisement to Facebook users who are currently living in Qatar but who, in the past, lived in India. The Facebook Marketing API provides estimates on how many monthly active users (MAUs) match these targeting criteria to facilitate better campaign and budget planning. We collect and visualize these estimates.


Facebook does not provide detailed information for how their targeting works. Previous academic work published by Facebook researchers uses both signals from a user’s self-declared “home city” as well as the current location of their social connections to infer a user’s country of origin. However, it is unclear if these or other features are used in the system currently in production at Facebook. This “black box” nature is a major limitation when using this type of data.
